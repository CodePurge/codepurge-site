Frequently Asked Questions

How do I install CodePurge?

CodePurge is available on the Mac App Store. Simply download and follow the on-screen setup instructions.

Does CodePurge permanently delete files?

In Test Mode, no files are deleted. In Live Mode, selected files are moved to the trash bin. Always review selections before proceeding.

Can I restore deleted files?

Once files are deleted in Live Mode, you can find them in your trash been and restore them as you would if you placed it in there yourself.

How can I report a bug or request a feature?

You can submit bug reports and feature requests on our GitHub Issues page.
